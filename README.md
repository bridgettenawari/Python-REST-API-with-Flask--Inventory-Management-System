> This is a CLI that intergrates CRUD, use of APIs and Python routing.
> It is an inventory manager.

~TIPS FOR RUNNING~
  > Run pipenv install
  > Run pipenv shell
  > Install flask using pip install flask
  > Install requests using pip install requests
  > pip install pytest

~TERMINAL COMMANDS~
  - python3 main.py show-items
  - python3 main.py add-item --name "name of item"
  - python3 main.py show-item --id "id number"
  - python3 main.py update-item --id "id number" --name "Updated Name"
  - python3 main.py delete-item --id "id number"
